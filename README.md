# Sistema de Cadastro de Clientes

Sistema web desenvolvido com CodeIgniter 4 para gerenciamento de clientes, permitindo operações completas de Cadastro, Consulta, Alteração e Exclusão (CRUD). O projeto utiliza integração com banco de dados MySQL e carregamento dinâmico de municípios a partir do estado selecionado.

## Funcionalidades
• Cadastro de clientes
• Validação de dados no formulário
• Consulta de clientes cadastrados
• Edição de registros
• Exclusão de clientes
• Seleção dinâmica de estados e municípios
• Integração com banco de dados MySQL
• Interface responsiva utilizando Bootstrap 5
• Feedback visual para operações realizadas
• Proteção CSRF nativa do CodeIgniter

## Tecnologias utilizadas
• PHP 8+
• CodeIgniter 4
• MySQL
• Bootstrap 5
• JavaScript (Fetch API)
• HTML5
• CSS3

# Instalação
1. Clone o repositório
2. Acesse a pasta do projeto
3. Crie um banco de dados MySQL e execute o script disponível no arquivo SQL
4. Configure as credenciais do banco no arquivo .env

# Executando o Projeto
• Abra um terminal na pasta public e execute: php -S localhost:8080
